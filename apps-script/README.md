https://docs.google.com/spreadsheets/d/1d7wQuQXtHht9O04eB_3d8OQiiHvUrx8iDq1uhtepIxI/edit

## สิ่งที่ระบบสร้างให้

เมื่อรัน `setupDatabase()` ระบบจะสร้างแท็บและหัวคอลัมน์อัตโนมัติ โดยไม่ต้องเตรียมคอลัมน์เอง:

- `Admins` — บัญชีเจ้าหน้าที่และรหัสผ่านแบบ hash
- `Events` — งานประชุม/กิจกรรม
- `Guests` — ผู้เข้าพักหนึ่งรายการต่อหนึ่งแถว
- `Settings` — ชื่อแอพ โลโก้ และคำอธิบาย

## ติดตั้ง

1. เปิด Google Sheet แล้วเลือก **ส่วนขยาย → Apps Script**
2. สร้างไฟล์ `Code.gs` และ `Index.html` จากโฟลเดอร์นี้
3. เปิด Project Settings แล้วเปิดการแสดงไฟล์ manifest จากนั้นใช้ `appsscript.json`
4. เลือกฟังก์ชัน `setupDatabase` แล้วกด Run
5. อนุญาตสิทธิ์การเข้าถึงชีต
6. ดู Execution log หรือค่าที่ฟังก์ชันส่งกลับ แล้วบันทึก Username/Password ชั่วคราวของ Super Admin
7. เลือก **Deploy → New deployment → Web app**
8. Execute as: **Me** และ Who has access: **Anyone**
9. เปิด URL ที่ Deploy ได้ แล้วเข้าสู่ระบบด้วยบัญชีชั่วคราว

> ควรเปลี่ยนรหัสผ่าน Super Admin หลังเข้าสู่ระบบครั้งแรก

## แบบ Excel ที่รองรับ

- RoomFlow Template ที่ดาวน์โหลดจากหน้าแอพ
- แบบตอบรับเข้าร่วมประชุมประเมินผล
- แบบตอบรับกรมวิทยาศาสตร์การแพทย์สัญจร ครั้งที่ 9/2568 (อุบล)

ตัวนำเข้าจะสร้างลำดับอัตโนมัติ รวมคำนำหน้ากับชื่อ แปลงประเภทการพักเป็นค่ามาตรฐาน และตัดรายการ “ไม่เข้าพัก” ออก โรงแรมและเลขห้องที่ไม่มีในแบบตอบรับสามารถกำหนดภายหลังในตารางแก้ไข

## ความปลอดภัย

- ไม่เก็บรหัสผ่านแบบข้อความธรรมดา
- ใช้ SHA-256 พร้อม salt
- Session หมดอายุภายใน 6 ชั่วโมง
- เจ้าหน้าที่ทั่วไปแก้ไขได้เฉพาะงานของตน
- Super Admin จัดการเจ้าหน้าที่และการตั้งค่าแอพได้
